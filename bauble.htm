A Pretty Bauble

A sphere can be parametrized using spherical polar coordinates , where the sphere is swept out in three dimensions by allowing two independent parameters θ and φ vary over a square of side 2π. If we pretend that the square is a torus instead, we can project a curve on the torus to the sphere. If we take a line which projects to a torus knot, that is, a line which wraps round the square m times in one direction and n times in the other (here m=4, n=17), we can project this line onto the sphere to obtain "spherical spirals", which make for some very pretty Christmas tree baubles!
